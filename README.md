# Potato-Disease-Classification
This project is a Deep Learning-based image classification system that detects potato leaf diseases from images. The model is trained to classify potato leaves into different disease categories using Convolutional Neural Networks (CNN).

---
## 📌 Project Overview
Potato crops are highly affected by leaf diseases such as:
- Early Blight
- Late Blight
- Healthy Leaves
This project helps in automatically identifying these diseases from leaf images, which can assist farmers and researchers in early disease detection and crop management.

---
## 🚀 Features
- Image classification using Deep Learning
- Trained on Potato Leaf Disease Dataset
- Detects multiple potato leaf conditions
- Easy to train and test
- Jupyter Notebook implementation

## ⚙️ Installation
Clone the repository:

```bash
git clone https://github.com/aryasmita1/Potato-Disease-Classification.git
```
Move into the project folder:

```bash
cd Potato-Disease-Classification
```
Install dependencies:

```bash
pip install -r requirements.txt
```

---
## ▶️ Usage
Run the Jupyter Notebook:

```bash
jupyter notebook
```

Open:

```bash
Potato Disease.ipynb
```

Train the model or test predictions directly from the notebook.

You can further improve performance by:

- Data augmentation

- Using transfer learning models like ResNet or EfficientNet
